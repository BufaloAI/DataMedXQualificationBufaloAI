# DataMedXQualificationBufaloAI
A folder of documents and explanations on the Machine Learning we built for the DataMedX Hackathon.

Our Aims are listed below:
- Create a data chart for one patient to show the progress that led to death over time.
- Plan and execute feature engineering to make the model ready for ML.
- Take the data, and create a Random Forest that predicts if someone will die or not.
- Create a method to give predictions on what should have been given to save the patient.
- Run that method in all time intervals for the patients, and see how to save them in those time intervals.

What We Achieved:
- Made the data parsable by our algorithm.
- Reduced data complexity using slope, mean, last_value, and count parameters.
- Created a Random Tree algorithm to predict weather a patient will die in a time period or not.
- Created two variants of that model (one relying on absolute death) to have better reliability.
- Created SHAP explanations to show what kills a patient.
- Created a website to publicly show that.
- Created a slide to assist with those.
